# 📊 Global Superstore Sales Analysis

## 📌 Project Overview

This project focuses on **Data Cleaning, Exploratory Data Analysis (EDA), and Data Visualization** using the Global Superstore dataset. The objective is to analyze retail sales data, identify business trends, and generate meaningful insights that can support data-driven decision-making.

This project was completed as part of the **DevRise AI & ML Internship – Task 1**.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Handle missing values and duplicate records
- Perform descriptive statistical analysis
- Explore sales, profit, customer, and regional performance
- Create informative visualizations
- Generate actionable business insights

---

## 📂 Dataset

**Dataset:** Global Superstore

The dataset contains over **50,000 retail orders** from more than **140 countries** and includes information such as:

- Order Date
- Ship Date
- Customer Details
- Product Category
- Sales
- Profit
- Discount
- Quantity
- Shipping Cost
- Region
- Country
- City

## 💡 Key Business Insights

- Technology products generated the highest revenue.
- Consumer customers contributed the largest share of sales.
- Standard Class was the most frequently used shipping mode.
- High discounts were generally associated with lower profits.
- A small number of products generated a significant portion of total sales.
- Sales varied across regions, highlighting differences in market performance.
- Some orders resulted in losses despite high sales, suggesting discount optimization opportunities.
